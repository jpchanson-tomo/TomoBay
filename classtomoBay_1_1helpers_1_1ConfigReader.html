static String [] tomoBay.helpers.ConfigReader.read ( String  pathToFile,
String  fileName 
)
static

reads a config file and passes back an list of strings representing different lines in the config file specified. It is important when constructing config files to put the correct setting on the correct line. line 1 = sandbox server string, line 2 = sandbox user token, line 3 = production server string, line 4 = production user token.

Parameters
pathToFilethe path to the file including the trailing slash
fileNamethe actual filename
Returns
List<String> containing the various lines of the config file. String[0] = blank, String[1] = sandbox server string, String[2] = sandbox user token String[3] = production server string, String[4] = production user token.
