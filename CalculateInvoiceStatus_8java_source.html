87  /**
88  * Takes the list of items generated in this.assembleItemsList and checks a PartList
89  * associated with each item against Stock held in winstock. For every item with enough
90  * stock level it adds a true value to a Set<Boolean> i.e. if everything is in stock the

92  * contain only 1 element = false, but if some items are in stock and others arnt it will

97  */
98  private Set<Boolean> checkStockLevelOfItems(List<String[]> itemsVar)
