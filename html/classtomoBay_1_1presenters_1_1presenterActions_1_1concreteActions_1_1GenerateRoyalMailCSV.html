Detailed Description

This class represents a presenter action that will generate a CSV file in a format compatible with the Royal Mail DMO, and will generate it in a place that is accessibe to the frontend.

The format of the input to this PresenterAction is: "Account - SalesRecNo - Service - Service Enhancement - format, "
