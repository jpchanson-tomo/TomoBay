Detailed Description

This class acts as the Invoker in a GoF style command pattern, it is responsible for the execution of AbstractWinstockCommands.

In this case it is also a factory and as such is responsible for the creation of these concrete command objects. The internal Enum defines the acceptable range of commands that can be created.
