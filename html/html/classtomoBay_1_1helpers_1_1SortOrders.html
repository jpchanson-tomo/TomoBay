Detailed Description

This class contains functionality for performing a category based sorting algorithm first ordering the data passed in by shipping type and the now sorted data by Pickeability.

this leads to data that is first displayed by pickeability then within each pickability type it is then sorted by shipping type.

each element of the list passed should be of the form: [orderID, buyerID, salesRecNo, shippingType, createdTime, picked, packed, shipped, invoiced]
