Detailed Description

This class provices functionality to provide pricing information on a compound item, i.e.

a parent item that is made of smaller composite items.

static final int [] prices ( int[]  childQuantities,
int[]  childCosts,
int  parentPrice 
)
static

This method returns an array of integers representing the prices of the individual child items.

VAT is not taken into account. the array returned will be the same size as the costs array as it assumes that you want the 1st element of the returned array to represent the individual price of the item represented by the first element of the costs array.

Parameters
childQuantitiesthe quantities of the items (in units)
childCoststhe cost values of the child items (in pennies)
parentPricethe cost of the parent item
Returns
int[] an integer array representing the prices of the items based on the cost array.
