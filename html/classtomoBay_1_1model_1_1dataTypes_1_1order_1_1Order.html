Detailed Description

This class represents an eBay order, and the relevant information associated with it.

Once instantiated this object will contain a Buyer object and an array of Transaction objects, and while this Order object will contain general information relevant to this eBay order, in order to gain more in depth details of the order i.e. buyer information or detailed information about the items that have been sold, it is necessary to query these contained objects.
