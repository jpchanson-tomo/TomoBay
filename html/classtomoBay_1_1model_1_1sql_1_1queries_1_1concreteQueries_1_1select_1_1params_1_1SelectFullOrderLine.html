Detailed Description

This class represents a query that provides the majority of information associated with a particular orderID as passed to the execute(HeteroFieldContainer parameter) method.

This query takes the following parameter:

  • OrdersTable.ORDER_ID

The query returns a List<HeteroFieldContainer> containing the following fields:

  • OrdersTable.ORDER_ID
  • OrdersTable.SHIPPING_TYPE
  • OrdersTable.SALES_REC_NO
  • OrdersTable.CREATED_TIME
  • OrdersTable.ORDER_TOTAL
  • OrdersTable.INVOICED
  • ItemsTable.ITEM_ID
  • ItemsTable.TITLE
  • ItemsTable.BRAND
  • ItemsTable.PART_NO
  • ItemsTable.NOTES
  • TransactionsTable.QUANTITY
  • TransactionsTable.PRICE
  • TransactionsTable.SHIPPING_COST
  • BuyerTable.BUYERID
  • BuyerTable.NAME
  • BuyerTable.STREET1
  • BuyerTable.STREET2
  • BuyerTable.CITY
  • BuyerTable.COUNTY
  • BuyerTable.POSTCODE
