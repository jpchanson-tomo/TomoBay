Detailed Description

This class represents an sql query that updates a record the buyer table in the database with new information.

This query requires the following parameters:

  • BuyerTable.NAME
  • BuyerTable.STREET1
  • BuyerTable.STREET2
  • BuyerTable.CITY
  • BuyerTable.COUNTY
  • BuyerTable.POSTCODE
  • BuyerTable.EMAIL
  • BuyerTable.PHONE
  • BuyerTable.BUYERID

These should be stored in a HeteroFieldContainer and passed to the execute(HeteroFieldContainer parameters) in order to run the query.
