Detailed Description

This object is responsible for providing the user with Insert or Update Query objects as specified by the internal enum, which defines all possible Insert or Update queries that can be executed on the database.

These Insert or Update query objects can either be created using the make() method or directly executed using the execute() method.

acceptable input values for those methods is defined in QueryType.
