static Set<Boolean> checkStockLevelOfItems ( List< HeteroFieldContainer >  itemsVar)
staticprivate

Takes the list of items generated in this.assembleItemsList and checks a PartList associated with each item against Stock held in winstock.

For every item with enough stock level it adds a true value to a Set<Boolean> i.e. if everything is in stock the set will only contain 1 element = true, if everything is out of stock the set will contain only 1 element = false, but if some items are in stock and others arn't it will contain 2 elements true and false.

Parameters
itemsVarthe list of items to check off against Stock levels
Returns
Set<Boolean> in stock - size=1 value={true}; out of stock - size=1 values={false}; partial stock - size = 2 values = {true,false}
