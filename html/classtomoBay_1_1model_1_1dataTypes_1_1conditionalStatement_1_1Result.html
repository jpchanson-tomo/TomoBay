Detailed Description

This class defines the Abstract Base class for all derived Result classes.

By implementing this class the user can define the true and false behaviours by encapsulating them within the appropriate methods(provided), the calling class (Conditional) will then invoke the correct behaviour when the Condition object gets evaluated.
