Detailed Description

This object defines a way of triggering services upon request, it contains a threadpool which is not user customiseable.

The default number of threads in the threadpool is 10, should this value need altering it will have to be done within the code.

if this fails it then waits until the second timeout has elapsed and tries an immediate shutdown. if both of these fail this method will return false
