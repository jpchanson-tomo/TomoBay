Detailed Description

This class represents a query that selects all data from the orders table that is not currently invoiced as determined by the value of the 'invoiced' field where a value of 0 indicates that the order is not invoicable, a value of 1 indicates that the order is partially invoicable and a value of 2 indicates that the order has been invoiced.
