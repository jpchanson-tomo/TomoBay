List<String[]> tomoBay.model.sql.queries.concreteQueries.SelectEbayItems.execute ( String[]  NotUsed) throws SQLException

execute the query

Parameters
NotUsedNOT USED
Returns
List<String[]> representing the results of the query. Each element in the list represents a row of the database and each element of the String[] represents a field.

The available fields for each element of the string[] are:

  • String[0] = itemID
  • String[1] = title
  • String[2] = sellCondition
  • String[3] = brand
  • String[4] = partNo
  • String[5] = noRequired
  • String[6] = cost
  • String[7] = notes
