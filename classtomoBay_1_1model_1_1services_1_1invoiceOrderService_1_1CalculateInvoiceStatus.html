Set<Boolean> tomoBay.model.services.invoiceOrderService.CalculateInvoiceStatus.checkStockLevelOfItems ( List< String[]>  itemsVar)
private

Takes the list of items generated in this.assembleItemsList and checks a PartList associated with each item against Stock held in winstock. For every item with enough stock level it adds a true value to a Set<Boolean> i.e. if everything is in stock the set will only contain 1 element = true, if everything is out of stock the set will contain only 1 element = false, but if some items are in stock and others arnt it will contain 2 elements true and false.

Parameters

Returns
